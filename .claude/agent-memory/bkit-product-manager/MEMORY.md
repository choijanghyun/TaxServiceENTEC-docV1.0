# Product Manager Agent - Memory

## Project: TaxServiceENTEC (경정청구 환급액 산출 시스템)

### 프로젝트 핵심 정보
- 목적: 법인세(CORP) + 종합소득세(INC) 경정청구 환급액 극대화 AI 기반 시스템
- 사용자: B2B 세무사/세무법인 (주), 향후 B2C 개인사업자 확장
- 기술스택: Spring Boot 3.x + Java 17 + PostgreSQL 15 + Clean Architecture 4계층
- DB: 83개 테이블, 107개 검증규칙, 53개 공식, 77개 점검항목(법인40+개인37)

### 핵심 설계 제약 (반드시 반영)
- 금액: 10원 미만 TRUNCATE (반올림 절대 금지)
- 비율: 소수점 3자리 TRUNCATE
- 원본 불변성: RI_S_RAW_DATA INSERT ONLY
- 세목 분기: TAX_TYPE=CORP vs INC (최저한세 계산 방식 완전 다름)
- 법인 최저한세: 과세표준 × 7~17%
- 개인 최저한세: 산출세액 × 35%/45%
- 적용순서: 감면 → 이월불가공제 → 이월가능공제

### 2025년 세법 개정사항 (요구사항 반영 필수)
- 창업+고용 중복배제: 2025년 이후 과세연도부터 택일 (이전은 병행 가능)
- 통합투자 추가공제율: 2025년부터 10% (종전 3%)
- 사회보험료공제 일몰: 2024.12.31. 종료 (2025년 이후 불가)
- 청년 판단기준: 2025년부터 근로계약 체결 당시 기준 (이전: 과세연도 말일)
- 창업감면 비청년: 2025년 이후 75% (종전 50%)
