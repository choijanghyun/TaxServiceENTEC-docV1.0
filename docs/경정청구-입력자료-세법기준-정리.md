## 1. 법인세 경정청구 - 주요 공제/감면별 필수 입력자료

### 1-0. 법인 공통 기본 정보 (모든 항목 공통 전제)

| 항목 구분 | 세부 입력자료 | 근거법령 | 수집방법 |
|---------|------------|--------|---------|
| 법인 식별 | 법인명, 법인등록번호, 사업자등록번호 | 법인세법 제2조 | 홈택스 자동 |
| 기업 규모 | 소기업/중기업/중견/대기업 (매출액·독립성 기준) | 조특법 시행령 제2조, 중소기업기본법 시행령 제3조 | 수동 (중소기업확인서) |
| 업종 | 한국표준산업분류(KSIC) 코드, 주업종/부업종 | 조특법 제6조, 제7조 | 홈택스 자동 |
| 본점 소재지 | 시·군·구 단위, 수도권/비수도권, 과밀억제권역 구분, 인구감소지역 여부 | 수도권정비계획법 제2조, 조특법 제7조 제1항 단서 | 수동 확인 |
| 사업연도 | 개시일/종료일 (결산월 확인) | 법인세법 제6조 | 홈택스 자동 |
| 과세표준 및 세액 | 각 사업연도 소득, 이월결손금, 과세표준, 산출세액, 결정세액, 납부세액 | 법인세법 제13조, 제55조 | 홈택스 자동 (기신고내역) |
| 경정청구 대상 | 대상 사업연도 (법정신고기한 후 5년 이내), 경정청구 사유 | 국세기본법 제45조의2 | 사용자 선택 |

> **법인 특화 핵심**: 본점이 수도권에 있으면 모든 지점을 수도권으로 간주 (조특법 제7조 단서). 단, 통합투자세액공제(제24조)는 실제 자산 설치 장소 기준으로 판단 (조특법 제130조).

### 1-1. 통합투자세액공제 (조특법 §24)

| 입력자료 구분 | 세부 항목 | 필수 증빙 | 근거조문 |
|------------|---------|---------|---------|
| **기업 요건** | 기업 규모 (중소/중견/대기업) | 중소기업확인서, 재무제표 | 조특법 §24, 시행령 §21 |
| **투자 자산 식별** | 자산 취득일, 취득가액 (세금계산서 기준) | 자산취득계약서, 세금계산서 | 시행령 §21 |
| | 자산 종류 (기계장치/설비/소프트웨어/산업재산권/운반구 등) | 검수조서, 등록증 | 시행령 §21 제1항 |
| | 시설 구분: 일반/신성장원천기술/국가전략기술 | 기술확인서 | 조특법 §24 제1항 각호 |
| **제외 대상 확인** | 중고품 여부, 리스자산 여부 (금융리스 포함), 수도권 과밀억제권역 내 여부 | 리스계약서, 매매계약서 | 조특법 §130, 시행령 §21 |
| | 소비성서비스업/부동산임대업 여부, 임대용 자산 여부 (2025년 이후 추가) | 사업자등록증 | 시행령 §21, 2024.12.31. 개정 |
| **추가공제 계산** | 직전 3년 평균 투자금액 (2021·2022·2023년 각각) | 과거 세무조정계산서, 재무제표 | 시행령 §21 제4항 |
| **공제율 결정** | 기업 규모별 공제율 (중소 10%/중견 3%/대 1%, 신성장 중소 12%, 국가전략 중소 16%) | — | 조특법 §24 제1항 |
| | 추가공제율: 2024년 이전 3%(국가전략 4%), 2025년 이후 10% | — | 2024.12.31. 개정, §24 제2항 |
| **사후관리** | 투자 완료 후 2년간 자산 처분/임대 여부 모니터링 | 자산대장 | 조특법 §146 |

**공제 계산 공식**:
```
공제액 = 투자금액 × 기본공제율
       + (투자금액 - 직전3년 평균투자) × 추가공제율
       [단, 추가공제분 > 0인 경우]
```

### 1-2. 통합고용세액공제 (조특법 §29의8)

| 입력자료 구분 | 세부 항목 | 필수 증빙 | 근거조문 |
|------------|---------|---------|---------|
| **기업 요건** | 기업 규모 (중소/중견/대기업), 적용기한(2028.12.31.) 이내 여부 | 중소기업확인서 | 조특법 §29의8, 시행령 §26의8 |
| **상시근로자 수 (핵심)** | 직전 사업연도 상시근로자 수 (연평균 = 매월 말일 합계 ÷ 12) | 고용보험 피보험자격 이력, 원천징수부 | 시행령 §23 제10항 |
| | 해당 사업연도 상시근로자 수 (동일 방식) | 고용보험 피보험자격 이력, 임금대장 | 시행령 §23 제10항 |
| **청년등 구분 (핵심)** | 청년(만 15~34세): 생년월일 및 병역이행 기간 | 주민등록등본, 병적증명서 | 시행령 §26의8 |
| | 장애인: 장애등급 및 등록일 | 장애인등록증, 장애인증명서 | 시행령 §23 제10항 |
| | 60세 이상 근로자 수 | 주민등록등본 | 시행령 §23 제10항 |
| | 경력단절여성: 퇴직 사유, 퇴직일, 재고용일 (퇴직 후 2~15년 이내) | 퇴직사유서, 재고용 근로계약서 | 시행령 §26의8 |
| | 북한이탈주민: 해당 증빙 | 하나원 수료증 등 | 시행령 §26의8 |
| **제외 대상 확인** | 최대주주·배우자·직계존비속 여부 | 주주명부, 가족관계증명서 | 시행령 §23 제10항 |
| | 임원(등기임원) 여부 | 법인등기사항전부증명서 | 시행령 §23 제10항 |
| | 단시간 근로자(월 60시간 미만) 여부 | 근로계약서 (소정근로시간) | 시행령 §23 제10항 |
| | 근로계약기간 1년 미만 여부 (단, 연속갱신 총 1년 이상 시 포함) | 근로계약서 전체 이력 | 시행령 §23 제10항 |
| | 파견근로자 여부 | 파견계약서 | 시행령 §23 제10항 |
| **추가공제 항목** | 정규직 전환: 전환 인원, 전환일, 전환 전 고용형태 | 기존+전환 근로계약서 | 시행령 §26의8 |
| | 육아휴직 복귀자: 복귀일, 복귀 후 6개월 이상 근무 이력 | 육아휴직 신청서, 복귀 확인서 | 시행령 §26의8 |
| **세액 산출** | 산출세액, 기존 적용 감면/공제 내역, 이월 미공제 세액 (직전 10년) | 이월세액공제명세서 | 조특법 §144 |
| **사후관리** | 공제 후 2년간 상시근로자 수 변동 이력 | 고용보험 가입이력 | 조특법 §29의8 제5항 |

**공제 계산 공식**:
```
[중소기업 2024 사업연도 기준]
공제액 = 청년등 증가인원 × 1,550만원
        + 일반 증가인원 × 850만원(수도권) 또는 950만원(비수도권)
        + 경력단절여성 재고용인원 × 1,300만원
        + 정규직 전환인원 × 1,300만원
        + 육아휴직 복귀인원 × 1,300만원

상시근로자 증가 = 해당 연도 - 직전 연도  [≤ 0이면 공제 불가]
청년등 증가분이 전체 증가분 초과 시 전체 증가분으로 한정
소수점 처리: 셋째 자리 절사 (반올림 절대 금지)
```

> **2025년 개정**: 청년 판단 시점이 "과세연도 종료일" → "근로계약 체결 당시"로 변경 (2025.1.1. 이후 최초 공제 사업연도부터). 2024년 이전 경정청구는 종전 기준 적용.

### 1-3. 창업중소기업 세액감면 (조특법 §6)

| 입력자료 구분 | 세부 항목 | 필수 증빙 | 근거조문 |
|------------|---------|---------|---------|
| **창업 요건** | 법인 설립일 (등기일) | 법인등기사항전부증명서 | 조특법 §6 제1항 |
| | 창업 형태 확인: 합병·분할·현물출자·사업양수·법인전환 해당 여부 | 법인등기사항전부증명서, 정관 | 시행령 §5 제2항 |
| | 최초 소득 발생 연도 (감면 기간 기산점) | 사업연도 손익계산서 | 조특법 §6 제1항 |
| **감면율 결정** | 대표이사 생년월일 (청년 여부: 만 15~34세) | 대표이사 주민등록등본 | 시행령 §5 |
| | 대표이사 병역이행 기간 (최대 6년 가산) | 병적증명서 | 시행령 §5 |
| | 본점 소재지 (수도권과밀억제권역 내/외, 인구감소지역 여부) | 법인등기사항전부증명서 | 조특법 §6 제1항 |
| | 벤처기업 확인 여부 및 확인일 (창업 후 3년 이내 시 50% 감면) | 벤처기업확인서 | 조특법 §6 제2항 |
| | 수입금액(매출액) 8천만원 이하 여부 (추가감면 제6조 제7항) | 손익계산서 | 조특법 §6 제7항 |
| **업종 확인** | 조특법 제6조 제3항 열거 감면 대상 업종 해당 여부 | 사업자등록증 (KSIC 코드) | 조특법 §6 제3항 |
| **감면 기간** | 최초 소득 발생 연도부터 5년 이내 여부 | 사업 개시 이후 결손금/소득금액 이력 | 조특법 §6 제1항 |
| **세액 산출** | 산출세액, 과세표준, 기존 신고 적용 감면/공제 내역 | 기신고 세무조정계산서 | 조특법 §6 |

**감면세액 계산 공식 (법인세)**:
```
감면세액 = 산출세액 × (감면 대상 사업소득 / 각 사업연도 소득) × 감면율

[감면율 체계]
- 2024년 이전 창업분:
  · 수도권과밀억제권역 외, 청년: 100%
  · 수도권과밀억제권역 외, 비청년: 50%
  · 수도권과밀억제권역 내, 청년: 50%
- 2025년 이후 창업분 (2024.12.31. 세법개정):
  · 수도권과밀억제권역 외, 청년: 100%
  · 수도권과밀억제권역 외, 비청년: 75% (종전 50%→75% 상향)
  · 인구감소지역: 100% + 최저한세 배제
```

### 1-4. 중소기업 특별세액감면 (조특법 §7)

| 입력자료 구분 | 세부 항목 | 필수 증빙 | 근거조문 |
|------------|---------|---------|---------|
| **기업 규모** | 소기업/중기업 구분 (감면율 차등), 적용기한(2028.12.31.) 이내 | 중소기업확인서, 재무제표 | 조특법 §7, 시행령 §6 |
| **업종** | KSIC 코드, 감면 제외 업종 (소비성서비스업, 부동산업 등) 해당 여부 | 사업자등록증 | 조특법 §7 제1항 각호 |
| **소재지** | 본점 소재지 (법인은 본점 기준 전체 간주): 수도권과밀억제권역/성장관리권역/수도권외 | 법인등기사항전부증명서 | 조특법 §7 제1항 단서 |
| **고용 현황** | 직전-해당 사업연도 상시근로자 수 (감면 한도 조정용) | 고용보험 피보험자격 이력 | 조특법 §7 제2항 |
| **감면 한도 계산** | 수입금액 (매출액) | 손익계산서 | 조특법 §7 제2항 |
| **감면세액 산출** | 산출세액, 감면 대상 소득 | 세무조정계산서 | 조특법 §7 |

**감면세액 계산 공식**:
```
감면세액 = 산출세액 × (감면 대상 사업소득 / 각 사업연도 소득) × 감면율

[감면율: 업종/지역/규모별]
- 제조업·광업 등 수도권 외: 소기업 30%, 중기업 15%
- 도소매업 수도권 외: 소기업 15%, 중기업 10%
- 수도권 과밀억제권역: 소기업 5%, 중기업 0% (사실상 배제)

[감면 한도]
- 연 1억원
- 상시근로자 감소 시: MAX(0, 1억원 - 감소 인원 × 500만원)
```

### 1-5. 연구인력개발비 세액공제 (조특법 §10)

## 5. 공통 계산 구조 및 핵심 주의사항

### 5-1. 세액 적용 순서 (법인세법 §59 / 소득세법 §60)

```
산출세액
  (1순위) - 세액감면 (§6 창업감면, §7 특별감면)
              → 농특세 비과세, 이월 불가 (최저한세 초과분 소멸)
  (2순위) - 이월공제 불가 세액공제
  (3순위) - 이월공제 가능 세액공제 (이월분 먼저, 당기분 나중)
              §24 통합투자, §29의8 통합고용, §10 R&D
              → 농특세 과세(§10 제외), 이월공제 10년
= 결정세액

주의: 최저한세 대상 공제감면 먼저 적용 후, 최저한세 배제 항목 나중 적용
```

### 5-2. 최저한세 비교표 (법인 vs 개인)

| 구분 | 법인 (조특법 §132 제1항) | 개인 (조특법 §132 제2항) |
|-----|----------------------|----------------------|
| 기준 | 과세표준 기준 | 산출세액 기준 |
| 중소기업 | 과세표준 × 7% | 산출세액 × 35% (3천 초과 45%) |
| 중견기업 | 과세표준 × 10~17% (초과누진) | 동일 (기업 규모 무관) |
| 대기업 | 과세표준 × 17% | 동일 |

### 5-3. 농어촌특별세 (농특세) 적용 여부

| 공제/감면 항목 | 농특세 | 실질 환급액 |
|-------------|-------|-----------|
| 창업중소기업 감면 §6 | **비과세** | 감면세액 100% |
| 중소기업 특별감면 §7 | **비과세** | 감면세액 100% |
| 연구인력개발비 §10 | **비과세** | 공제세액 100% |
| 통합투자 §24 | 과세 (20%) | 공제세액 × 80% |
| 통합고용 §29의8 | 과세 (20%) | 공제세액 × 80% |
| 사회보험료 §30의4 | 과세 (20%) | 공제세액 × 80% |
| 외국납부세액공제 | 비과세 | 공제세액 100% |

### 5-4. 상호배제(중복 적용 불가) 핵심 매트릭스 (조특법 §127 제4항)

| 조합 | 2024년 이전 | 2025년 이후 |
|-----|-----------|-----------|
| §6 창업감면 + §7 특별감면 | **택1 (불가)** | **택1 (불가)** |
| §6 창업감면 + §24 투자 | **불가** | **불가** |
| §6 창업감면 + §29의8 고용 | **가능** | **불가 (택1)** |
| §6 창업감면 + §30의4 사보 | **불가** | §30의4 일몰로 해당없음 |
| §7 특별감면 + §24 투자 | **불가** | **불가** |
| §7 특별감면 + §29의8 고용 | **가능** | **가능** |
| §7 특별감면 + §30의4 사보 | **가능** (§127 단서) | §30의4 일몰로 해당없음 |
| §24 투자 + §29의8 고용 | **가능** | **가능** |
| §24 투자 + §10 R&D | 동일 자산 **불가** | 동일 자산 **불가** |
| §10 R&D + 모든 감면 | **가능** (§127 배제 대상 아님) | **가능** |

### 5-5. 환급가산금 계산

```
환급가산금 = 환급세액 × 이자율 × (경과일수 / 365)

이자율: 국세기본법 시행규칙 §19의3 (2025년 기준 연 3.1%, 시중금리 연동 변동)
기산일: 납부일 다음날 (법인세) / 납부일 다음날 (종합소득세)
종료일: 환급 결정일

법인: 환급가산금은 익금불산입 (법인세법 §18 제4호)
개인: 환급가산금은 비과세 소득 (소득세법 열거주의)
1원 미만 절사 (국세기본법 §86)
```

### 5-6. 금액 계산 원칙 (절사 원칙)

| 계산 항목 | 처리 방법 | 근거 |
|---------|---------|-----|
| 모든 세액 | 10원 미만 절사 (반올림 절대 금지) | 국세기본법 §86 |
| 환급가산금 | 1원 미만 절사 | 국세기본법 §86 |
| 상시근로자 수 | 소수점 셋째 자리 절사 (둘째 자리까지) | 시행령 §27의4 제6항 |
| 공제율/배분비율 | 소수점 셋째 자리 절사 | 시행령 §23 제10항 |
